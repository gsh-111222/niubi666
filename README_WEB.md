# 万向灵蛇网页版（MVP）

## 1. 安装依赖

```bash
pip install -r requirements.txt
```

## 2. 启动

```bash
python web_app.py
```

启动后浏览器访问：

`http://127.0.0.1:5000`

## 3. 已支持功能

- 方向键按住控制，松开自动发送停止
- 模式切换（蛇形/小车）
- 速度调节、角度调节、重置
- 设置下位机 IP/端口并应用
- 开启/停止 UDP 接收
- 浏览器显示实时视频流（MJPEG）

## 4. 说明

- 底层协议与原 `snake_backend.py` 一致，网页端复用了同一套控制逻辑。
- 这个版本是可运行骨架，后续可继续做手机端布局优化、按钮连发防抖、断线提示等。
